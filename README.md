Приложение для управления абонентами компании "Телебор"

--- О приложении ---

- Мобильное приложение для управления данными абонентов, тарифами и заявками в техподдержку. Приложение поддерживает три роли пользователей: обычный пользователь, техподдержка и администратор.

--- Функциональность ---

- Для всех пользователей:

	- Вход в приложение по логину и паролю;

	- Интуитивно понятный интерфейс с карточками;


- Для обычного пользователя:

	- Просмотр своих данных (ФИО, адрес, телефон);

	- Редактирование своих данных;

	- Просмотр текущего тарифа;

	- Смена тарифа через выпадающий список;

	- Создание заявок в техподдержку;


-Для техподдержки:

	- Просмотр всех заявок от абонентов;

	- Отображение email отправителя и даты заявки;

	- Просмотр полного текста заявки;


- Для администратора:

	- Просмотр списка всех абонентов;

	- Редактирование данных любого абонента;

	- Изменение тарифа любого абонента;



--- Технические особенности ---

- Язык программирования: Java

- Минимальная версия Android: API 21 (Android 5.0)

- Используемые библиотеки:

	- AndroidX AppCompat и CardView



--- Установка и запуск ---

1. Клонируйте репозиторий

2. Откройте проект в Android Studio

3. Соберите проект (Build -> Make Project)

4. Запустите на эмуляторе или реальном устройстве
